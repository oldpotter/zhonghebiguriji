# 【复谷注意事项】

沅汐：辟谷21天左右的人在复谷的第一个星期内吃的一定要单一，切记，吃的单一再单一，一点点增加。

书因㜣 ：好，食材上单一。调料、酱料 味道上需要单一么？ 第二天了，胃依然没什么饿的感觉，只是嘴巴想吃味道。

沅汐：味道上也要单一简单。

书因㜣 ：第三天开始，可以蔬菜水果添加了么？

沅汐：    尽量不加，到七天之后再加。

解释一下。每一种蔬菜身上的元素都是差别巨大的，以前你们要吃复合蔬菜，原因是身体已经无法分辨这些信息了，反而种类越多补充越多。 但是现在是不一样的，你们的信息的干净的，忽然接触复合蔬菜的信息身体会处理不来，会导致很多不消化、浮肿甚至发炎的现象。 你们现在的身体不需要过多营养补充，因为现在的身体是最健康的状态。恢复饮食只是恢复一个习惯，让胃进点东西开始蠕动就行。这一周里如果出现了哪一餐满足不了饮食条件，就选择喝水千万不要求其次。

茶慢竹：我复食4、5天了。这两天腿浮肿厉害。是盐摄入多了吗？

沅汐：是的，吃的太复杂了。

茶慢竹：啊！我大部分是生食，不吃炒菜了。看来还是复杂了。

沅汐：单一是一种蔬菜，千万别复合蔬菜。

茶慢竹：早上打了五谷汁，看来也得少喝了。

茶月开 ：红糖的那个方子还要继续喝吗？

沅汐：那个就没必要了。

境千：我瘦了好多，太阳穴都凹陷了，精神也不好，人有点虚。

沅汐：嗯，喝红糖水，喝糯糯的粥。

境千：辟谷之后身体是一个健康的状态。但自己看上去并不健康，是期间练功和静心没到位，是这样吗？

沅汐：那是身体在调整病还没反完，以后再来，第一次这样行了。

书因㜣：那人岂不是一直不吃更好，就一直保持这个最健康的状态，饮食反而是个满足口欲的负担了？

沅汐：也不是这样，我以前指导过一个辟谷半年的，人一直不吃会对生命没有欲望，饮食是为了生活下去。

 书因㜣：里面好多学问啊。 我自己感受是，现在好热爱生活，热爱厨房，买菜洗澡做菜，收纳打扫。这些生活里最最基本的事。这以前在我看来简直是浪费时间，不得已而为之。

书意童：我这两天大脑一直是懵的，好像动不动，脚有点浮肿，是不是也是吃多了吃杂了？

沅汐：是的，吃的简单下来，你这是身体累了。

书因㜣：复谷期间，身体是否还是会继续调整的？会持续上一段时间，如果饮食上注意的话。因为这两天早上醒来，脚踝那里冒凉气，两条腿发凉，自己觉的是在排寒气。

沅汐：会的，如果每天活的很当下，这样的调整会一直持续下去，可能由强到弱，但会持续。

书因㜣 ：嗯嗯，明年继续好好辟谷。

书赞：刚复谷，我今天做了一次铺灸应该没事吧。

沅汐：那没事。

书洵美：之前有个朋友断食治好了忧郁症。

沅汐：那是断食让人清静了，内心不杂乱忧郁症好了。 得看断多久。

茶念伶：复食后发现不喜热食，甚至不太爱熟食，炒菜之类的， 喜欢吃偏生冷 ，有条件的情况下更喜欢吃新鲜蔬果，油腻炒菜辟谷期间很馋，但是复食后反而不想吃，吃的时候也不如想象中享受， 是生熟食有个能量落差的问题吗？

沅汐：是的，人的本能是食生的，这样吃的最环保。

书因㜣 ：是本能？和能量有关系么？

沅汐：吃生食是最保护胃和营养的。

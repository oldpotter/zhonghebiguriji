# 7月6日、7月7日

**19:46**

沅汐：    咱们这个是辟谷，换句话可以说外面的很多辟谷都不一定有咱们这个辟谷效果好。

现在大多数人还都认为辟谷是不吃，其实那是最表象。吃的种类越少越能起到切断以往信息的效果，但每人状态不一，切合实际状态才是最有效的方法，不能死磕，辟谷的方式也要循序渐进。

书赞：我下午做吐纳，不停打嗝吐痰，后来吐出酸水，刺痛喉咙，有灼热感。这是胃酸吗？

沅汐：那是排出多余胃酸了，多喝水，继续吐纳。

境千：我这馋的欲望来的好猛，之前没有，就差舔屏。我不知道啥叫觉知了？就是知道一下自己现在有个馋的欲望吗？我记得你说知道一下它就没了 来了再知道一下。我问呢越绝欲望越强。

沅汐：谁告诉你立刻没了，立刻没了你得有那个功力啊。 这跟任何一个情绪一样他一定会越来越强然后脱离。 仍然不管，只是觉知。 你现在还在管它强弱，这个也不管。 静心的时候只管静心，静心就是看你能不能不着念在任何一个外在发生，所有的发生跟你的静无关。

境千：我妇科从第一天开始到现在，每天都有排时多时少，没问题吧？

沅汐：排的现象说一下。

境千：11天，深褐色，量时多时少，每次小便时还会随之排出一些小碎杂质的感觉。

沅汐：    一点没事，艾灸绒塞肚脐，每天6个多小时拿掉，敲打腹股沟让它快点。

还有个方法快点，你买艾灸泥，加热之后，一定要热一点，敷在尾椎骨和尾椎附近，会很难受，但是非常快。

书因㜣：这几天不爱喝甜的，就想喝咸的，酱油汤可以喝么，对咸菜汤特别渴望。

沅汐：辟谷期间所有自己起欲望想吃的都要去觉知，让后让这个脱离，不能吃。

茶月开：三次米汤感觉好有压力，米汤是否可以不喝？

沅汐：可以，你身体不想要了就减，以水代替，本来到了第三个七天也是安排全部是水了。

茶子参：喝米汤和红糖水喝到一半不想喝，微微恶心，是因为身体不需要了。和胃不好没关系吧？

沅汐：你喝水恶心吗？

茶子参：不恶心，一点没事。

沅汐：那是不需要了。

茶念伶：今天一天软趴趴辟谷以来最虚弱的一天。连采能都采不动，也不想吃东西，喝米汤纯粹是口舌满足喜欢咸味，肚子一点不 饿，可是人很软，东西都拿不动，刚才给我先生做饭也完全不想吃。

茶念伶：水昨天昨天喝很多口渴的不行。今天喝了两瓶就喝不下去了，红糖水也不想喝，想尝咸味，静心感觉人是浮的，沉不下去。

沅汐：是排病反应，一碗红糖水一点点喝下去，吃一点点坚果，然后静心。

无耕道：我一说话就口干，舌头好像没有分泌任何口水一样，必须喝一小口水才能继续下去，这个情况好几天了。

沅汐：口干是辟谷的普遍排病现象，身体里的水分在产生转化，没事。

茶见隐：我左侧的头疼持续四天了，每天下午五点多开始，我就看着它疼，不抗拒，现在是在头顶到额头里面有一段。

沅汐：继续觉知，这些都会排掉。

书末青：我这两天胸闷气短，是排病吗？要做点什么？

沅汐：你肺本来就虚，也是在排，覃中刮痧。

书因㜣：什么是血液的转换，几次听到你用这个词了。

沅汐：    固有信息转换的时候人会有许许多多现象出来，红血球里的、水分子里的、氧分子里的、神经上的……

转换过程就是固有信息一层层脱离，取而代之的是新鲜干净充满活力的信息。 脱离之前会有很严重的现象，跟情绪脱离很像，脱离之前情绪浮现的很大，还是觉知就掉落了。

茶念伶：月经是出血症吗？是可以避免的吗？野生动物不来月经，圈养以后有月经，饮食也有关系吧？

沅汐：    不能这么说，只能说人长大了，有了成年人的种种因素，月经就回来。 是很正常的生理现象，繁衍后代一定要有生理期。只要有胎盘的动物都得来，只是周期不一样。

我最近看了一个不完全统计的调查，平均值里最晚来月经的人是22岁，最早9岁，跟每个人心性成熟程度有很密切的关系。 很多运动型的人来的晚，她们思维较简单。

茶见隐：我今天感觉自己差点挂了，早晨户外采光时，突发低血糖，眼前顿时一片发黑，感觉整个世界都黑暗了，意识清醒，想走动一下，结果完全没有方向感，向左摔在了树丛里，周围没有人。想了想，起来坐在地上，一会儿，世界慢慢明亮起来，由于北京昨天一天下雨，左侧的衣裤全是泥，好在没有摔坏，也没有划伤自己，走回家去倒是挺有精神的。

沅汐：喝红糖水吃一点坚果一个橙子，静心练功，你的头还在恢复。
